# lazydocker-web
> Run `lazydocker` in a Docker container and serve output over http using `ttyd`

![screenshot](./docs/screenshot.png)


## Run
- `git clone https://github.com/mattpowell/lazydocker-web.git`
- `cd lazydocker-web`
- `docker compose up -d`
